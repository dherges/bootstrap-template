Bootstrap Template

A minimalistic project set-up using npm, bower, and grunt to quickly kick-off Bootstrap-based projects. In addition, phantomjs can be used to run and report QUnit test suites.

$ npm install
$ node_modules/.bin/bower install
$ npm install -g grunt-cli
$ grunt
